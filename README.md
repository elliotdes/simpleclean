# simpleclean

A simple & clean Hugo theme.

## Features

- Tags
- [picocss](https://picocss.com/)
- Google Analytics
- Comments ([giscus](https://giscus.app/))
- Mermaid diagrams

## Installation

In your `hugo.toml` set:

```toml
theme = "simpleclean"
```

## Configuration

Make sure to configure Google Analytic and Giscus in your projects `hugo.toml`.
